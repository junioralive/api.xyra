Welcome to Xyra API

A high-performance, scraping-based REST API designed to serve metadata, streaming links, and images for manga, anime, and dramas.

Why use the API?

While Xyra is awesome to use directly, developers love building custom experiences. We provide the raw, structured data you need to build your own apps, bots, or integrations.
